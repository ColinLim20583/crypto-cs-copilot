# Account Security and 2FA

## Two-factor authentication (2FA)
All accounts must enable at least one 2FA method. Supported methods, from
strongest to weakest: hardware security key (FIDO2/passkey), authenticator app
(TOTP), and SMS. We recommend a passkey or authenticator app; SMS is
vulnerable to SIM-swap attacks.

## Lost 2FA device
1. On the login page choose "Security verification unavailable" → "Reset 2FA".
2. Verify identity with a government ID and a live face check.
3. Reset requests are reviewed within 24 hours. After a successful reset,
   withdrawals are automatically suspended for 48 hours as an anti-theft
   measure. This hold cannot be removed early under any circumstances.

## Anti-phishing code
Set an anti-phishing code in Security settings. The code appears in all
genuine emails from the platform. Emails without your code should be treated
as phishing and reported.

## Account compromise — immediate steps
If you suspect unauthorized access:
1. Freeze the account immediately: Security → Disable Account, or via the
   in-app emergency freeze.
2. This suspends withdrawals, trading, and API keys instantly.
3. Contact support from your registered email to begin the recovery review.
4. File a police report if funds were moved; support will provide transaction
   records for law enforcement upon official request.

## What support will never ask
Support staff will NEVER ask for your password, 2FA codes, seed phrases,
private keys, or ask you to transfer funds to a "safe account". Anyone doing
so is a scammer. Report impersonation attempts through the official report
form.

## Device and address management
Review authorized devices under Security → Device Management and remove
unknown devices. Enable withdrawal address allowlisting so funds can only be
sent to pre-approved addresses, with a 24-hour delay for new additions.
